Using news.gmane.org

An alternative to subscribing to each mailing list, is to use the newsgroup (available in modern email clients such as Thunderbird or Outlook) nntp://news.gmane.org, and subscribing to the gmane.comp.java.jbehave.* newsgroups.

Using web archives
